## SAT Words
> Common SAT Words

## Install
```
$ npm install --save sat-words
```

## Usage
```javascript
var SATWords = require('sat-words')

SATWords.getWords(function(words){
  console.log(words)
})
```

## API
### .getWords(callback)
```javascript
var SATWords = require('sat-words')

SATWords.getWords(function(words){
  console.log(words)
})
```
